Project Terbaru

Beberapa karya yang saya kembangkan dengan fokus pada tampilan menarik, dan fungsionalitas.

Sistem Manajemen Data Pendaftaran (BIMBEL) dengan Dashboard Admin Responsif

Aplikasi web fungsional yang dikembangkan menggunakan framework Flask (Python) dan database MySQL. Proyek ini berfungsi sebagai sistem satu pintu untuk mengelola data pendaftaran siswa baru dan pendaftaran ulang, serta menyajikan data tersebut dalam Dashboard Admin yang dilindungi oleh autentikasi login. Desain antarmuka dibuat bersih, modern, dan responsif untuk memastikan kemudahan navigasi. Dilengkapi dengan fitur unduh laporan data dalam format PDF

Sistem E-commerce Fashion Lengkap dengan Administrasi Produk dan Integrasi Pembayaran Midtrans

Pengembangan Aplikasi E-commerce menggunakan PHP Murni dan MySQL yang fokus pada pengalaman pengguna yang intuitif (UI/UX Design) dan fungsionalitas transaksi yang lengkap.

Fitur Utama:

  1. Sistem Administrasi (Admin Panel): CRUD (Create, Read, Update, Delete) untuk Produk dan Kategori.
  2. Sistem Transaksi Real-Time: Fungsionalitas Keranjang Belanja dan proses Checkout.
  3. Integrasi Pembayaran: Implementasi payment gateway Midtrans untuk memproses pembayaran secara aman dan real-time.
  4. Dashboard Keuangan: Menyajikan ringkasan statistik dan laporan keuangan sederhana.

Landing Page Toko Roti (Roti ManisKu)

Pengembangan Website Statis (Front-End) untuk toko roti (Bakery Landing Page) menggunakan HTML5 dan CSS.
